# SQL Visual Learning Tool

Interactive web application for learning SQL through step-by-step visual execution.

## Features

- **4 Progressive Datasets**: Basic to expert level
- **Step-by-Step Execution**: See how SQL transforms data
- **30+ Example Queries**: Learn by example
- **Custom Query Mode**: Practice your own SQL

## Datasets

1. **Customers & Orders** - Basic JOINs
2. **Books & Borrowers** - Intermediate queries
3. **Employees & Departments** - NULL handling & aggregations
4. **Search Trends** - Time-series data (Google BigQuery style)
